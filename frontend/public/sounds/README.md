# Notification Sounds

This folder contains notification sounds for the application.

## Required Sound Files

You need to add the following MP3 files to this folder:

1. **message.mp3** - Sound played when receiving a new message
2. **call.mp3** - Sound played for incoming calls (loops)
3. **notification.mp3** - General notification sound
4. **send.mp3** - Sound played when your message is sent (optional)
5. **receive.mp3** - Sound played when receiving a message in chat (optional)

## Where to Get Free Sounds

Here are some resources for free notification sounds:

### Free Sound Resources:
- **Freesound.org**: https://freesound.org/search/?q=notification+sound
- **ZapSplat**: https://www.zapsplat.com/music/notification-sounds/
- **SoundBible**: http://soundbible.com/categories/Notification_Sounds_22.html
- **Mixkit**: https://mixkit.co/free-sound-effects/notification/

### Recommended Sounds:
- **Message**: Short, pleasant "ding" or "pop" sound
- **Call**: Ringing tone or phone ring sound
- **Notification**: Subtle "beep" or "chime"

## How to Add Sounds

1. Download MP3 or WAV files from the resources above
2. Convert to MP3 if needed (use online converters)
3. Rename to match the file names above
4. Place in this `sounds` folder
5. Refresh your browser

## Note

Until you add actual sound files, the notifications will work but won't play any sound. The app will not break - it will just log a console message if sounds are missing.
